ToDo List Functionality

Use the input box to type which task you want to add to the ToDo list. Clicking the Add button will add it to the list. Clicking the Delete Last Entry will delete the last entry of the list. Clicking Display will display the entire list below. As an optional feature, the delete specific entry button has been added along with a selection option. Select which entry you wish to delete specifically then click Delete Specific Entry.
